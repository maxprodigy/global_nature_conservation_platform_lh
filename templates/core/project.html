  OUR PROJECTS  

" With currently 15 projects under our sponsorships, Global Nature Conservation has been a hub for campaigns that are dedicated to conserving, preserving, and restoring the ecosystems for the past few years. "

1000 Tap Water Project

This project is focused on eliminating the demand for single-use plastic water bottles by building a network of 1000 drinking fountains throughout hilly region of Musanze.


Bumbogo Sanitation Programme

This project was done last year and it helped village in rural areas to help them build portable toilets in their house and surroundings. This programme decreased number of water borne disease cases significantly.


Clean Lake Jen Project

Clean Lake Jen Project, a project by Global Nature Organization, made mass awareness and focused on cleaning the Jen village lake in Karim-Lamido, Taraba State. This project reduced the waste thrown into the lake each year by approximately 1 Metric Ton.


Animal Rescue

There are lot of cases of wildfire each year throughout Nigeria and biodiversity is heavily affected by it. Many animals are forced to come into human settlement due to this wildfire. This project helped to capture them and move them back to dense forests.


Green Project

Green Project is an afforestation programme that had a goal of planting 10k trees throughout Africa. It is still an ongoing project and so far about 4k trees have been planted.


Sanitation Project

This project was entirely was due to help of volunteers. It focused on cleaning the areas of Kimironko axis. This project took place for 7 days with the help of several volunteers.


Wild animals Treatment

An innovative idea of collecting the data and taking a survey of wildlife in Yankari National Park. This project helps the sick animals and also have a record of their species which again helpts to increase their population.


Poach Not Project

The Poach Not Project is a wildlife conservation programme that has a goal of reducing poaching by 13% by 2030 throughout Africa. It is still an ongoing project and so far significant efforts has been made to create awareness.
